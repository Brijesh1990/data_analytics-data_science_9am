# what is SQL ?
  1. SQL stands for structured query language 
  2. SQL is a case-insenstive language
     examples : insert | Insert | INSERT 
  3. SQL is used to create database structured and table structured. 
  4. SQL create a structured type of data 
 
     **examples of structured type of data**

     1. table | CSV | excel formate is structured type of data
   5. SQL create structured type of data
   6. SQL is provides some query or commands   
   
   **SQL query or commands**
   There are some types of sql query 
   1. DDL (data definition language)
   2. DML (data manupulation language)
   3. DQL (data query language)
   4. TCL (transanctional query language)

   **DDL (data definition language)**

   1. DDL is used to create database structured 
      **syntax of create database structured**
      **examples**
      ```
        create database databasename
        or
        create database companydb;
      ``` 
   2. DDL is also used to create a table structured
